- ## 如何启动
	- 下载并解压.
	  logseq.order-list-type:: number
	- 执行 `./cpolar authtoken ${authtoken}`
	  logseq.order-list-type:: number
	- 执行 `./cpolar http ${port}`
	  logseq.order-list-type:: number
		- 执行之后会启动 http 和 https 两种协议的域名映射.

- ## Use a package
	- 如何使用一个 Package:
		- 创建一个  `pubspec` 文件 : `pubspec.yaml`
		  logseq.order-list-type:: number
		- `pubspec.yaml` 中添加自己需要用到的 `Package` 依赖, 和其它的一些 `Metadata` (比如: 版本号) .
		  logseq.order-list-type:: number
		- 执行 `dart pub get` 获取依赖的 `Package` .
		  logseq.order-list-type:: number
		- `import` Package 中的自己需要的 library , 并调用自己需要的代码. (参见: [[Dart Library & Import]] )
		  logseq.order-list-type:: number

- ## 什么是 FVM
	- 即 Flutter Version Management, 是 Flutter SDK 的版本管理工具.
- ## FVM 的功能
	- 支持给每个项目设置不同的 Flutter SDK 版本.
	  logseq.order-list-type:: number
	- 支持即时切换 Flutter SDK 版本.
	  logseq.order-list-type:: number
